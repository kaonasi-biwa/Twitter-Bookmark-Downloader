# Twitter Bookmark Downloader

ブックマークはダウンロードできません。
いずれ対応します。

## What's this?
![image](https://github.com/kotonefami/Twitter-Bookmark-Downloader/assets/69312738/bac20382-c8c6-4dd6-9400-200b468cdf52)

画像にダウンロードボタンが追加されます。

## Install
ビルドしてください。
```shell
npm run build
```
```shell
pnpm run build
```
```shell
node ./scripts/build.mjs
```
お好きなコマンドをどうぞ。

## Compatibles
Google Chrome は確認済み。Firefox は未検証。
